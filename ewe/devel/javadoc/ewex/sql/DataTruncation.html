Constructor Details

DataTruncation

public DataTruncation(int index,
                      boolean parameter,
                      boolean read,
                      int dataSize,
                      int transferSize)
This method initializes a new instance of DataTruncation with the specified values. The descriptive error message for this exception will be "Data truncation", the SQL state will be "01004" and the vendor specific error code will be set to 0.
Parameters:
index - The index of the parameter or column that was truncated.
parameter - true if a parameter was truncated, false otherwise.
read - true if a data column was truncated, false otherwise.
dataSize - The original size of the data.
transferSize - The size of the data after truncation.
