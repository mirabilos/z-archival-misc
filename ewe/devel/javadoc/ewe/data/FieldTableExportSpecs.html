public class FieldTableExportSpecs
extends LiveObject
This is used by a FieldTableModel to export table data to the clipboard or to a file. It als provides an Editor which the user can use to specify particular details about the export. You can execute ewe ewe.data.FieldTableExportSpecs to see what it looks like.
