EditableObject
public interface EditableData
extends DataUnit, Textable, Transferrable
An EditableData is a simple data object that can create and utilize it's own Control for editing. Similar to the LiveData concept, but targeted at simpler data objects. An EditableObject is an implementation of this interface.

toSaveableData

public void toSaveableData(Wrapper data)
This asks the object to set the Wrapper to be a value that can be used to save the state of this object. The value set in the Wrapper should be
  • Any primitive value.
  • A ewe.sys.Time object for date/time values.
  • A String object for complex data.
  • A ByteArray or byte[] object for more complex data.
If you are calling setObject() for one of the specified types, you must NOT provide a null object.
