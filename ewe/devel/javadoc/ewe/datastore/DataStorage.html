Constructor Details

DataStorage

public DataStorage(File file,
                   String mode)
            throws IOException,
                   IllegalArgumentException
Open or create a DataStorage file. If the mode indicates read-write ("rw") AND the file does not exist, then it will be created and initialized as a new DataStorage file.
Parameters:
file - The data storage file.
mode - must be "r" or "rw". An illegal argument exception will be thrown if it is neither.
Throws:
IOException - if there is an error opening or creating the DataStorage file.

DataStorage

public DataStorage(File file,
                   String mode,
                   DataProcessor decryptor,
                   DataProcessor encryptor)
            throws IOException,
                   IllegalArgumentException
Open or create a DataStorage file. If the mode indicates read-write ("rw") AND the file does not exist, then it will be created and initialized as a new DataStorage file.
Parameters:
file - The data storage file.
mode - must be "r" or "rw". An illegal argument exception will be thrown if it is neither.
decryptor - An optional decryptor - necessary for reading the file if it is encrypted.
encryptor - An optional encryptor - necessary for writing to the file if it is encrypted.
Throws:
IOException - if there is an error opening or creating the DataStorage file.

DataStorage

public DataStorage(RandomAccessStream stream)
            throws IOException
Open or create a DataStorage file in a RandomAccessStream. If the stream length is zero a new DataStorage will be created in the stream (if it is open for read-write). Otherwise, the existing data is used.
Parameters:
stream - The stream to use.
Throws:
IOException - if there was an error creating or reading the DataStorage.

Parameters:

DataStorage

public DataStorage(String file,
                   int mode,
                   boolean create)

Deprecated. - use one of the Constructors that throw an IOException instead.

Creates a new DataStorage and opens up the specified file. If the file does not exist and create is true, it will create a new one, otherwise it will open it in the specified mode (RandomAccessFile.READ_ONLY or READ_WRITE). After constructing you should check the isValid() flag to determine if it is valid.
