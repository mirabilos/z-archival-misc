  • java.lang.Object
    • ewe.datastore.DatabaseEntry
Known Direct Subclasses:
DataEntryData
public class DatabaseEntry
extends Object
Each field value is stored as follows in the byte array. byte 1 = ID; A value of 0 indicates an invalid or deleted field. byte 2 = Reserved; byte 3,4 = Length of Data; A short integer value indicating the length of the data. byte[length of data] - The data itself.
