public class IntChoice
extends mChoice
An IntChoice is a mChoice that treats setInt() and getInt() differently. Normally getInt() on an mChoice will return the chosen index. However getInt() on an IntChoice will convert the String at the chosen index to an integer value and return it. The same goes for setInt().
