Class MD5

  • java.lang.Object
    • ewe.security.MD5
public class MD5
extends Object
This class implements the "RSA Data Security, Inc. MD5 Message-Digest Algorithm
It has been ported from a public domain Java implementation (by Jon Howell) to Ewe by Tom Arn (www.t-arn.com)

To compute the message digest of a chunk of bytes, create an MD5 object 'md5', call md5.update() as needed on buffers full of bytes, and then call md5.digest(), which will fill a supplied 16-byte array with the digest. Get the digest with md5.getDigest() or md5.toString()
