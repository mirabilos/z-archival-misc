Using The Zip Library

Ewe provides a Zip file implementation with an API similar to the Java API. These are in the ewe.zip package and the main classes that you would use are:

ü      DeflaterOutputStream – This is used to compress a data stream.

ü      InflaterInputStream – This is used to uncompress a deflated stream.

ü      ZipInputStream – This is used to read individually compressed files within a single ZIP file.

ü      ZipOutputStream – This is used to write individually compressed files to a single ZIP file.

ü      ZipFile – This is used to randomly access compressed files within a single ZIP file.

ü      ZipEntry – Every file within a ZIP file has one of these associated with it.

 

The DeflaterOutputStream and InflaterInputStream objects are only used for storing compressed data inside non-ZIP files (e.g. within a PNG image). Please consult the APIs for each of these to see how to use them.

 

Another API is provided in ewe.util.zip. This provides different ZipFile, ZipEntry, ZipInputStream and ZipOutputStream implementations that are almost identical to the java.util.zip package. This is based on the functionality of ewe.zip but makes it easier to port Java code to and from Ewe.

 

The example code below uses the ewe.util.zip package.

Writing Data to a Zip File

Use a ZipOutputStream to create a new ZIP archive. You must give it a destination output stream to write to when you construct it. Then for each compressed file you want to store you must first create and store a zip entry for the file, and then write the individual bytes for the file into the zip stream. If you are writing a STORED entry, you must set the size of the ZipEntry (using setSize()) and the CRC of the entry (using setCRC()) before you write the bytes.

 

Here is an example of creating a new zip file:

 

package samples.zip;

import ewe.util.zip.*;

import ewe.io.*;

 

//##################################################################

public class CreateZip{

//##################################################################

 

//===================================================================

public static void addEntry(String name,int method,File f,ZipOutputStream zos) throws IOException

//===================================================================

{

     ewe.sys.Vm.debug("Adding: "+f);

     ZipEntry ze = new ZipEntry(name);

     ze.setMethod(method);

     if (method == ZipEntry.STORED){ //Must set these for STORED entries.

           CRC32 crc = new CRC32();

           crc.reset();

           byte [] buff = new byte[1024];

           Stream is = f.toReadableStream();

           while(true){

                int got = is.read(buff,0,buff.length);

                if (got <= 0) break;

                crc.update(buff,0,got);

           }

           is.close();

           ze.setCrc(crc.getValue());

           ze.setSize(f.getLength());

     }

     zos.putNextEntry(ze);

     Stream is = f.toReadableStream();

     new IOTransfer().transfer(is,new StreamAdapter(zos));

     is.close();

}

 

//===================================================================

public static void main(String args[]) throws IOException

//===================================================================

{

     ewe.sys.Vm.startEwe(args);

     ZipOutputStream zos = new ZipOutputStream(new File("Sample.zip").toWritableStream(false));

     addEntry("First.java",ZipEntry.DEFLATED,new File("samples/ui/ShowExec.java"),zos);

     addEntry("Second.java",ZipEntry.STORED,new File("samples/ui/Events.java"),zos);

     addEntry("Third.java",ZipEntry.DEFLATED,new File("samples/ui/TestPanel.java"),zos);

     zos.finish();

     zos.close();

     ewe.sys.Vm.debug("Done.");

     ewe.sys.mThread.nap(1000);

     ewe.sys.Vm.exit(0);

}

//##################################################################

}

//##################################################################

 

Reading Data from a Zip File

Use a ZipInputStream to read the entries from a ZIP archive. You must give it a source input stream to read from (e.g. a FileInputStream). Once the stream is open, you use getNextEntry() to get the individual zip entries. You would then read each byte from the ZipInputStream until it reports that it is at the end of the stream.

 

Here is an example which read data from a Zip file.

 

package samples.zip;

import ewe.util.zip.*;

import ewe.io.*;

 

//##################################################################

public class ReadZip{

//##################################################################

 

//===================================================================

public static void main(String args[]) throws IOException

//===================================================================

{

     ewe.sys.Vm.startEwe(args);

     ZipInputStream zis = new ZipInputStream(new File("Sample.zip").toReadableStream());

     while(true){

           ZipEntry ze = zis.getNextEntry();

           if (ze == null) break;

           byte [] buff = new byte[512];

           int total = 0;

           while(true){

                int got = zis.read(buff,0,buff.length);

                if (got <= 0) break;

                total += got;

           }

           ewe.sys.Vm.debug(ze.getName()+" = "+total+" bytes.");

     }

     zis.close();

     //

     // Use a ZipFile to do the same.

     //

     ZipFile zf = new ZipFile("Sample.zip");

     ewe.util.Enumeration it = zf.entries();

     while(it.hasMoreElements()){

           ZipEntry ze = (ZipEntry)it.nextElement();

           ewe.sys.Vm.debug(ze.getName()+" Uncompressed:"+ze.getSize()+" Compressed:"+ze.getCompressedSize());

     }

     ewe.sys.mThread.nap(3000);

     ewe.sys.Vm.exit(0);

}

 

//##################################################################

}

//##################################################################

 

 

Using the ZipFile Object

A ZipFile object will automatically parse an entire ZIP file and build up a table of entries. You can then access entries individually.

 

You would open the ZIP using one of the constructors e.g.:

ZipFile zf = new ZipFile(new File("Sample.zip"));

 

Calling getEntries() will give you an Iterator object which you can use to step through each ZipEntry in the file. You can also call getEntry(String name) to get an entry which matches the specified name.

 

Once you have a ZipEntry you can read the data associated with it by calling getInputStream(ZipEntry entry). This returns a stream from which the original data can be read.
