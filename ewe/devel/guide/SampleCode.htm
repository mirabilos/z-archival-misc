Sample Code

The Ewe SDK comes with a directory called samples under the main classes directory. This contains a number of sample Ewe programs. The classes in bold listed below are the classes that you should run. For example:

 

ewe samples.ui.TestPanel

 

samples.ui – Basic Control use and Layout

 

samples.ui.TestPanel – Demo for simple control layout.

samples.ui.TestForm – Demo for Form display.

samples.ui.TestSingleContainer – Demo for SingleContainer control.

samples.ui.TestTabForm – Demo for using tabs in a Form.

samples.ui.TestAppForm – Demo for using the AppForm control.

samples.ui.Events – Demo for GUI event handling.

samples.ui.FirstLayout – Demo for CellPanel layout.

samples.ui.ShowExec – Demo for using show() and execute() methods.

samples.editor – Use of the Editor control

 

samples.editor.EditPerson – Demo for editing data using FieldListener.

samples.editor.LoginScreen – Demo of an Editor editing its own data.

samples.data – Use of the LiveData and LiveObject   classes

 

samples.data.PersonInfo – A LiveObject implementation.

samples.data.BigPersonInfo – An extension of PersonInfo.

samples.progress – Use of the ProgressBarForm  classes

 

samples.progress.ProgressDemo – A demo of using ProgressBarForms.

samples.mosaic – Use of the Mosaic and InteractivePanel classes

 

samples.mosaic.MosaicDemo – A demo of an InteractivePanel with custom cursor.

samples.paint – Using Graphics and Canvas

 

samples.paint.Painter – A mini paint application.

samples.sockets – Using Sockets and InetAddress

 

samples.sockets.SocketDemo – A demo for connecting and using sockets.

samples.sockets.LookupForm – A demo for name resolution.

samples.zip – Using the Zip library

 

samples.zip.CreateZip – A demo for creating a Zip file.

samples.zip.ReadZip – A demo for reading a Zip file.

samples.locks – Using Lock objects

 

samples.locks.Buffer – A demo for using a Buffer object.
